## and-game

```JS

// Input
andGame(['idli', 'sambhar', 'chutney]);

// Output
idli, sambhar and chutney.


// Input 
andGame(['idli', 'sambhar', 'chutney], true);

// Output
idli, sambhar & chutney.

```

## Parameters
1. Array of elements.
2. Boolean value if you wanna use '&' instead of 'and'. By default it is set to false.
